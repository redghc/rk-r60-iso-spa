## Software (VIA)

According to your keyboard **PID E480**, it is an **R65 wired mode with RGB**. Because this version supports **QMK/VIA**, the RK driver is not used. Here are the steps for using VIA:

1. Open the VIA official website: [usevia.app](https://usevia.app)
   — or download the desktop app from: [github.com/the-via/releases](https://github.com/the-via/releases/releases)

2. Obtain the configuration file specific to your keyboard (`RK-R65.json`):
   [Download RK-R65.json](https://drive.google.com/file/d/1K2FlNWwpZiZZgRP9s0xlzZ24zVvytb2q/view?usp=sharing)

3. Click on **SETTINGS** to access the settings page and enable the **"Show Design tab"** option. This will display the **DESIGN** page.

4. Go to the **DESIGN** page and click on **"Load"**. Select the `RK-R65.json` configuration file from the dialog box and open it.

5. A pop-up device dialog will appear in the top left corner. Click on the device to establish a connection.

You can then explore the full customization of your keyboard.
